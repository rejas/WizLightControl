Wiz Light Control v1.0
================================
Control a Philips Wizlight with Mac Menu Bar.

Please set the IP of the bulb before use.

Requirements
==========
- macOS 13.0

Functions
==========
- Turn On / Off
- Set Brightess
- Set Color Temperature
- Change Scene
- Set Speed
